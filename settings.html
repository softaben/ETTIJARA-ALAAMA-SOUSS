Instructions pour GitHub

Pour configurer l'intégration GitHub:

  1. Créez un nouveau dépôt sur GitHub nommé "ettijara-alaama-souss"
  2. Générez un token d'accès personnel dans les paramètres GitHub
  3. Ajoutez les permissions "repo" et "workflow" au token
  4. Copiez-collez le token dans le champ ci-dessus
  5. Cliquez sur "Tester la Connexion" pour vérifier

Instructions pour Firebase

Pour configurer Firebase:

  1. Allez sur Firebase Console
  2. Créez un nouveau projet "ettijara-alaama-souss"
  3. Ajoutez une application Web à votre projet
  4. Copiez la configuration et collez-la ci-dessus
  5. Dans la section "Realtime Database", créez une base de données
  6. Configurez les règles de sécurité pour autoriser l'accès
